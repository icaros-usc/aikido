bool aikido::planner::ompl::GeometricStateSpace::StateType::mValid

Indicates whether the state has been initialized to represent a valid state.

This allows samplers to indicate failure by setting this flag to false when sampling fails. The StateValidityChecker should check this flag when determining if the state is valid. The value defaults to true.
